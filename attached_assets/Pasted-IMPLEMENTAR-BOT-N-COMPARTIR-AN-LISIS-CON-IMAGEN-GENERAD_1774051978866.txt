IMPLEMENTAR BOTÓN "COMPARTIR ANÁLISIS" CON IMAGEN GENERADA

OBJETIVO:
Permitir al usuario compartir una imagen visual del análisis (viralidad)

---

1. AGREGAR BOTÓN EN RESULTADO

Ubicación: debajo del score

<button className="btn-share" onClick={generarImagen}>
  📤 Compartir análisis
</button>

---

2. USAR HTML2CANVAS

Instalar:

npm install html2canvas

Importar:

import html2canvas from "html2canvas";

---

3. CREAR CONTENEDOR PARA CAPTURA

<div id="share-card" className="share-card">
  <h2>CoinCash</h2>
  <p>Seguridad TRON en tiempo real</p>

  <div className="score">{score}/100</div>
  <div className="risk">{nivelRiesgo}</div>

  <div className="wallet">{wallet}</div>

  <p className="footer">Verifica antes de enviar dinero ⚠️</p>
</div>

---

4. FUNCIÓN GENERAR IMAGEN

const generarImagen = async () => {
const element = document.getElementById("share-card");

const canvas = await html2canvas(element);
const image = canvas.toDataURL("image/png");

const link = document.createElement("a");
link.href = image;
link.download = "coincash-analisis.png";
link.click();
};

---
